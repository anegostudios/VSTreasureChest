# VSTreasureChest

This mod show cases the World Gen API of Vintage Story by adding treasure chests beside trees. It also adds a /treasure command to place a treasure chest in front of the player. A walkthrough of how this mod was created can be found at http://wiki.vintagestory.at/index.php?title=WorldGen_API

Please be aware that the project makes use of the environment variable VINTAGE_STORY which needs to point to the installation location.
